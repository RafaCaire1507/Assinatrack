<h1>Template padrão da Aplicação</h1>

<p>
  O layout padrão do site foi construído com as linguagens de marcação HTML e CSS, 
  e a linguagem de programação JavaScript foi utilizada para a construção do menu responsivo.
</p>

<p>
  As páginas terão como elementos padrões o menu de navegação, o header e o footer, 
  além dos elementos de identidade visual.
</p>

<p>
  O código utilizado para a construção dos elementos citados, incluindo a responsividade,

  As imagens e ícones utilizados no projeto estão disponíveis

</p>

<h3><b>Tela de Minhas Assinaturas</b></h3>
<p>
  Exibe a lista de assinaturas organizadas por categorias, permitindo ao usuário visualizar 
  detalhes como valor, data de vencimento, forma de pagamento e status. Também apresenta 
  o valor total aberto no mês e oferece ferramentas de busca e adição de novas assinaturas.
</p>
